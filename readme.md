# PureOil - Oil Mill E-commerce Website

PureOil is a modern web application for oil mills, providing an intuitive platform where users can purchase a variety of oils and vendors can list their products for sale. The project is built using [Vite](https://vitejs.dev/) and [React](https://react.dev/), delivering fast performance and a highly interactive user experience.

## Features

- **User Portal**: Browse, search, and purchase oils through a clean shopping interface.
- **Vendor Portal**: Vendors can register and list their oil products for sale, manage inventory, and track orders.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Fast & Scalable**: Developed using Vite for lightning-fast builds and React for scalable UI components.
- **Secure Authentication**: Login and registration for both users and vendors.
- **Order Management**: Users can view order history, and vendors can manage orders received.

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/ShivamMathtech/pureoil.git
   cd pureoil
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```

4. **Visit the app**
   Open [http://localhost:5173](http://localhost:5173) in your browser (default Vite port).
